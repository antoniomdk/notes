# PYTHON

## Packages

### Functional Programming

* Missing features of functional programming: [fn.py](https://github.com/fnpy/fn.py)
  * Scala-style lambdas
  * Inmutable data structures
  * Streams (lazy sequences)
  * Trampoline decorator
  * Function wrapper for composition and partial application
  * Currying
* Data processing inspired by Scala, Spark and Microsoft Linq: [PyFunctional](https://github.com/EntilZha/PyFunctional)
  * Chainable operators
  * Parallel processing
  * Streams (lazy sequences)
  * Support for csv, json, sqlite, gzip... 
* Haskell-style Lenses: [Python-lenses]( https://github.com/ingolemo/python-lenses)
* Data abstractions: [PyMonad](https://pypi.python.org/pypi/PyMonad/)
  * Functor, Applicatives and Monad operators
  * Predefined monoids (int, list, str)
  * Predefined monads (Maybe, Either, List, Reader, Writer, State)
* Design By Contract: [PyContracts](https://github.com/AndreaCensi/contracts)
* Property-based Testing: [Hypothesis](https://github.com/HypothesisWorks/hypothesis)

### Concurrency and Parallelism

* Akka Actor model in Python: [Pykka](https://github.com/jodal/pykka)
